Welcome to BP Pulse

Track your blood pressure over time to identify patterns and maintain a healthy heart.

Manual entry works without any setup. Photo entry requires a little bit of setup and once complete you can capture blood pressure (BP) data directly from a picture of your BP device.

To do this, you need an Anthropic API key, stored only in your browser and sent only to api.anthropic.com.

How to get one
  1. Sign in at console.anthropic.com
  2. Add a payment method (if you haven't already)
  3. Settings → API Keys → Create Key
  4. Paste the key below (it starts with sk-ant-)
Add or change the key at any time via the top right menu → API Key
